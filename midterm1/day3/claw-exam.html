Question 6 · Chance Error Check 6 pts

An SFO dashboard samples n = 120 flights uniformly at random (SRS) and observes 78 on-time departures.

(a) [2]

Compute the sample proportion p̂ of on-time flights. Report as a decimal to two places.

Show solution

p̂ = 0.65

78 / 120 = 0.65. This is the point estimate of the true on-time rate.

(b) [2]

Using the plug-in formula, compute SE(p̂) = sqrt(p̂(1 − p̂)/n). Round to three decimal places.

Show solution

SE ≈ 0.043

p̂(1 − p̂) = 0.65 × 0.35 = 0.2275. Divide by 120 → 0.0018958. Square root ≈ 0.0435.

(c) [2]

If we quadruple the sample size to n = 480 (same population proportion), what happens to SE? Choose one.

Show solution

Answer: Cuts in half.

SE ∝ 1/√n. Increasing n by factor 4 multiplies the denominator by √4 = 2, so SE halves.

Question 7 · Sampling Flash 2 pts

A ride-hailing lot has 40% electric vehicles. You sample 3 cars uniformly at random with replacement.

What is the probability at least one of the sampled cars is electric?

Show solution

Answer: (C) 0.784

“At least one” = 1 − P(no EV). P(no EV in one draw) = 0.6. With replacement, P(no EV in three draws) = 0.6³ = 0.216. So 1 − 0.216 = 0.784.

Quick Formulas

• Sample mean: x̄ = (1/n) Σ x_i
• Sample variance: s² = (1/(n−1)) Σ (x_i − x̄)²
• SE for sample proportion p̂ from SRS: SE = sqrt(p̂(1 − p̂)/n)

Pandas Reminders

• .loc[label] is label-based; .iloc[pos] is position-based.
• groupby().agg('max') collapses to one row per group.
• filter keeps original rows for groups whose boolean expression is True.
