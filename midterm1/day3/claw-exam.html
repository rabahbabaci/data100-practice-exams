Question 5: Taxi-Out Inference Discipline 6 Pts

An SRS of n = 360 departures records average taxi-out time \bar x = 16.2 minutes with sample SD s = 5.8. A bootstrap run gives SE = 0.31 minutes for estimating the population mean.

(a) [2 pts]
Construct a 95% confidence interval using z^* = 1.96. Show your arithmetic.
Show Solution

CI = 16.2 ± 1.96 · 0.31 = 16.2 ± 0.6076 → (15.59, 16.81) minutes when rounded to two decimals.

(b) [2 pts]
If a new SRS of the same population quadruples the sample size (all else equal), what happens to the standard error?
Show Solution

Answer: Shrinks by a factor of 2.

SE scales like 1/√n; quadrupling n halves the SE.

(c) [2 pts]
Operations proposes texting every sampled pilot twice if they ignore the first survey link. Which bias is this targeting?
Show Solution

Answer: Non-response bias.

Reminder nudges more sampled individuals to answer; it does nothing about how the frame was chosen.

Question 7: Logistic Gradient Check 5 Pts

Binary outcomes follow logistic regression with p_i = σ(θ_0 + θ_1 x_i), where σ(z) = 1 / (1 + e^{-z}). The negative log-likelihood is L(θ) = -\frac{1}{n}\sum_{i=1}^n [y_i \log p_i + (1 - y_i) \log(1 - p_i)]. Choose the correct expression for ∂L/∂θ_1.

Show Solution

Derivative: \(\frac{1}{n}\sum (p_i - y_i)x_i\).

The chain rule gives ∂p_i/∂θ_1 = p_i(1-p_i) x_i, which collapses into the familiar residual term.
